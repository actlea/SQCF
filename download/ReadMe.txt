========================================================================
    控制台应用程序：download 项目概述
========================================================================

应用程序向导已为您创建了此 download 应用程序。

本文件概要介绍组成 download 应用程序的每个文件的内容。


download.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 
    的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

download.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。 
    它包含有关项目文件与筛选器之间的关联信息。 在 IDE 
    中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。
    例如，“.cpp”文件与“源文件”筛选器关联。

download.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h，StdAfx.cpp
    这些文件用于生成名为 download.pch 的预编译头 (PCH) 文件和
    名为 StdAfx.obj 的预编译类型文件。

/////////////////////////////////////////////////////////////////////////////
其他注释：

应用程序向导使用“TODO:”注释来指示应添加或自定义的源代码部分。
Dict.txt
	这是表格的属性项。
downlaod.cpp
	该文件中函数的主要功能是下载网页，其中还包含有判断网页格式和转码的函数。
Table.cpp
	该文件主要的功能是提取各类型表格中的数据。
unity.h
	这是公共变量的声明文件。
option.cpp
	主要包含辅助型的函数。
InitLink.txt
	存放的初始种子链接信息，可以自行添加和修改，不同的种子，有不同的操作。
	需要注意的是，所有的种子链接都不包含协议字段。
	wiki提取表格数据
	national提取球员数据和国家队信息
	pic提取世界杯32强国家队的图片
	china特意找到的中国国家队球员肖像照
